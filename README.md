# Engineering_Practice_Project

工程实践项目《校园信息智能查询智能体》

主要是利用爬虫 + 智能体 + Vue技术栈，在网页端发送给智能体自己的校园账户及密码，实现以下功能：1.自动登录学校的教务系统；2.在教务系统中查询成绩；3.在教务系统中查询课表；4.在教务系统中查询教室，最终返回给用户相应的查询信息

主要包含以下文件：  
1.爬虫文件 campus_crawler  
简介：采用Python构建，主要面向于cuit的教务处，可以进行登录、爬取课表、爬取成绩，并且对爬取的html文件进行解析，然后返回相应的信息

使用方式：修改config/setting.py文件中的账户和密码为自己真实的账户和密码，然后运行crawler/cuit_crawler.py文件即可进行登录、爬取课表、爬取成绩，并对爬取的html文件进行解析，然后返回相应的信息
